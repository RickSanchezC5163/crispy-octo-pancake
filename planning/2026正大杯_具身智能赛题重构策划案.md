# “正大杯”第十六届全国大学生市场调查与分析大赛
## 具身智能方向赛题重构策划案（可执行版）

> 目标：把“具身智能发展趋势”从宏观技术叙事，重构为**可在300–800份问卷内完成、可做3–5个统计模型、能落到企业营销/产品决策**的市调赛题。

---

## 第一部分：去年报告拆解（基于《正文.docx》）

### 1.1 报告结构框架（目录逻辑）
去年报告整体是典型“市调大赛”结构：
1. 问题综述（背景/意义/文献）
2. 研究思路与创新
3. 调查方案（对象、问卷、抽样、质控）
4. 预调查与量表检验（信效度）
5. 正式调查与现状分析
6. 客户特征模型
7. 满意度影响因素模型（PLS-SEM）
8. 潜在客户识别与细分（聚类+预测）
9. 结论与策略建议

这个逻辑的优点是：从“现象描述”逐步推进到“机制解释”和“策略落地”，评委读起来会感觉路径完整。

### 1.2 调研方法体系提炼

**（1）样本与抽样**
- 采用分层+多阶段抽样（区域→地市/区县→农业从业者）
- 有预调查（先小样本修订问卷，再正式调查）
- 线上+线下混合发放，辅以追访，控制无效问卷和偏差

**（2）问卷设计**
- 基础人口属性
- 使用/非使用分流题
- 使用动机与行为题
- 满意度量表题（潜变量-观测变量体系）
- 未来意向与建议题

**（3）统计模型链路**
- 信度/效度检验（Cronbach α、KMO、Bartlett）
- 客户特征模型（二元选择/Logistic）
- 满意度机制模型（PLS-SEM）
- 潜客细分（谱聚类）
- 潜客转化预测（XGBoost）

> 这套方法从“描述”到“解释”再到“预测”，是一个比较有竞争力的组合。

### 1.3 去年方案优势
1. **结构完整**：研究问题—方法—结论—策略闭环明显。  
2. **数据方法层次高**：不仅有基础统计，还有SEM、聚类、机器学习，提高“分析深度感”。  
3. **策略导向明确**：结论对应政府、企业、推广路径，具备应用价值。  
4. **有预调查与量表检验**：体现规范性，能提高评委信任。

### 1.4 若参加本届比赛可能短板
1. **模型“过重”风险**：若样本质量一般，复杂模型容易出现“解释华丽但稳健性不足”。
2. **业务问题聚焦不够尖锐**：涉及现状、满意度、潜客、发展路径多个目标，可能导致主线分散。
3. **“可执行商业动作”还可更细**：例如缺少“企业下一季度如何投放、给谁投放、预算如何分配”的强决策颗粒度。
4. **技术名词较多**：若答辩表达不够“业务语言化”，评委可能觉得更像方法展示，而非市场决策支持。

## 第二部分：判断“具身智能方向”是否适合市调赛

### 2.1 结论先行
“具身智能”**可以做市调赛题**，但必须从“技术趋势研究”转成“明确市场决策问题”。

### 2.2 四项适配性判断

1. **是否符合市场问题导向？**  
- 原始“具身智能发展趋势”偏宏观，不够市场化。  
- 若改成“某场景下客户是否购买/租赁具身智能设备、愿付价格、关键阻碍”就符合。

2. **是否有明确调研对象？**  
- 可以明确，如“连锁餐饮门店店长/运营负责人”“中大型商超运维负责人”“医院后勤负责人”。  
- 对象必须是**有采购建议权或使用决策影响力**的人。

3. **是否可问卷化/访谈化？**  
- 可以。可问卷化维度包括：认知、使用场景、痛点、风险感知、信任、ROI预期、支付意愿、采购条件。  
- 可访谈化维度包括：部署阻力、售后顾虑、系统对接难点、组织内审批流程。

4. **是否具备数据分析深度？**  
- 具备。至少可做：因子分析、Logit回归、聚类、联合分析/价格敏感度、SEM（择其3-5个）。

## 第三部分：将今年方向重构为“可参赛可落地”选题

## 3.1 推荐最终选题（缩小版）
### 选题标题（可直接用于报名）
**《从“替人”到“增效”：连锁餐饮后厨清洁场景中具身智能机器人采购意愿与支付阈值研究——基于华东地区门店运营决策者的实证调查》**

> 该题满足：
> - 明确行业：连锁餐饮
> - 明确细分人群：门店店长/区域运营/后厨管理者
> - 明确决策问题：是否采购（或租赁）+能接受什么价格/服务条件
> - 可用300-800份问卷
> - 可用3-5个模型完成

## 3.2 研究对象界定
- **行业边界**：连锁餐饮（正餐、快餐、茶饮三类连锁体系）
- **地域边界**：华东（建议沪苏浙皖，便于样本组织）
- **样本对象**：门店店长、后厨主管、区域运营（要求至少参与设备采购建议）
- **样本规模**：
  - 正式问卷：N=450~600（建议目标550）
  - 预调查：N=50~80
  - 深访：15~25人（用于构建问卷与解释模型结果）

## 3.3 核心研究问题（5条）
1. 当前后厨清洁作业的核心痛点是什么（人效、稳定性、卫生标准、流失率）？
2. 哪些因素显著影响门店对具身智能清洁机器人的采购/租赁意愿？
3. 不同门店类型（快餐/正餐/茶饮、单店/区域连锁）在需求上是否存在可识别细分？
4. 在不同服务方案（买断/租赁/维保打包）下，门店可接受支付阈值是多少？
5. 厂商若要提高转化率，应优先优化“功能、价格、售后、培训”中的哪些要素？

## 3.4 调研方法设计（问卷+统计模型）

### A. 问卷模块（可执行）
1. **门店画像**：门店类型、面积、日单量、后厨人数、人工成本占比
2. **现状痛点**：清洁频次、耗时、人员流动、卫生扣分风险
3. **技术认知与信任**：对机器人稳定性、安全性、可维护性的认知
4. **方案偏好**：买断/租赁/服务包偏好、最低可接受性能要求
5. **支付意愿**：价格区间、回本周期要求、预算审批阈值
6. **采购意愿结果变量**：3个月内试用意愿、12个月内采购概率

### B. 建议采用的3-5个模型（推荐5个）
1. **信度效度检验**：Cronbach α、KMO、Bartlett（保证量表可用）
2. **探索性因子分析（EFA）**：提炼“效率收益感知、风险感知、服务信任”等潜因子
3. **二元Logit回归**：解释“是否愿意在12个月内采购/租赁”的关键影响因素
4. **K-means聚类**：形成3-4类门店画像（如“成本敏感型”“效率优先型”“品牌形象型”）
5. **价格敏感度模型（PSM或Van Westendorp）**：估算可接受价格带和最优报价区间

> 如团队统计能力较强，可将EFA替换或补充为PLS-SEM；但建议优先保证“业务解释清晰”。

## 3.5 数据分析工具建议
- **SPSS**：描述统计、信效度、EFA、Logit（易上手）
- **R/Python**：聚类、可视化、稳健性检验
- **SmartPLS（可选）**：如使用PLS-SEM
- **Excel/BI看板**：将结果转化为“厂商行动清单”

## 3.6 创新点设计（区别普通问卷报告）
1. **“意愿+价格阈值”双输出**：不仅知道“想不想买”，还知道“多少钱能买”。
2. **场景化任务拆解**：把“后厨清洁”拆为地面清洁、油污处理、闭店清洁等子场景，形成可执行产品路线。
3. **细分市场打法**：基于聚类给出分人群策略，不是一刀切建议。
4. **模拟投放回报**：用样本估计“每100家目标门店可转化多少家”，直接对应企业营销预算。

## 3.7 竞争力评估（评委视角）

### 优势
- 题目有现实感：餐饮人工成本、流失、卫生监管都是真问题。
- 数据路径清晰：可通过门店负责人问卷快速拿到有效样本。
- 商业落点明确：结果可直接用于机器人厂商定价与渠道投放。
- 模型难度适中：既有统计深度，又不至于“炫技失控”。

### 风险与应对
- **风险1：样本不够“决策人”** → 问卷筛选题设置“是否参与设备采购建议”。
- **风险2：受访者对具身智能理解不足** → 问卷开头用统一场景图文说明。
- **风险3：价格题乱填** → 采用分层价格题+异常值剔除规则。

## 第四部分：一次“选题定稿会议”规划（90分钟）

## 4.1 会议目标
在一次会议内完成：
1. 最终选题标题确认
2. 研究对象和样本框确认
3. 核心研究问题（3-5条）确认
4. 模型组合确认（3-5个）
5. 问卷框架与分工确认

## 4.2 讨论顺序（建议时间）
1. **10分钟：赛事要求对齐**（必须是市场决策问题，不做宏大趋势）
2. **20分钟：候选选题PK**（每个选题回答“行业-人群-决策问题-数据可得性”）
3. **20分钟：样本与数据路径确认**（样本来源、数量、发放渠道、回收节奏）
4. **20分钟：模型组合确认**（保底模型与进阶模型）
5. **10分钟：问卷框架确认**（变量表、量表来源、筛选题）
6. **10分钟：里程碑与分工**（预调查时间、正式调查时间、答辩材料负责人）

## 4.3 关键分歧点（提前准备）
1. 做“多行业泛具身智能”还是“单行业深挖”？（建议后者）
2. 做“技术性能评价”还是“采购决策研究”？（建议后者）
3. 模型上“PLS-SEM优先”还是“Logit+聚类+价格模型优先”？（建议后者，稳）
4. 样本追求“全国分散”还是“区域可控高质量”？（建议区域可控）

## 4.4 每个人准备任务（会前24小时）
- **队长**：准备赛事规则约束清单（哪些不能做）
- **方法负责人**：准备3套模型组合方案（基础版/标准版/进阶版）
- **问卷负责人**：准备一级指标与题项池（含量表来源）
- **渠道负责人**：准备可触达样本名单与预计回收量
- **行业负责人**：准备5页以内行业事实卡（市场规模、痛点、竞品）

## 4.5 会议最终产出文件（必须落地）
1. 《选题定稿单》（最终标题、研究边界、核心问题）
2. 《样本与回收计划表》（目标样本、渠道、时间、备选方案）
3. 《模型路线图》（变量定义、模型顺序、输出指标）
4. 《问卷V1大纲》（筛选题+核心量表+结果变量）
5. 《项目排期甘特图》（预调查→正调→分析→报告→答辩）
